# Network Intrusion Detection System (NIDS)

A Python-based Network Intrusion Detection System developed using Scapy.

## Features

- Packet Sniffing
- Intrusion Detection
- Suspicious IP Monitoring
- Port Scan Detection
- Colored Terminal Logs
- TXT Log Export
- CSV Log Export
- Live Statistics Counter
- Real-time Monitoring

## Technologies Used

- Python
- Scapy
- Colorama
- CSV Logging

## Installation

```bash
pip install scapy colorama
